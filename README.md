Battleship
==========

This project is for demonstration purpose.
This is an implementation of the game Battleship for one player.

Installation
------------

```bash
$ composer install
```

Run the game
------------

Launch the web server
```bash
$ php -S localhost:8080 -t web web/index.php
```

Browse the url `http://localhost:8080/`

Run tests
---------

```bash
$ vendor/bin/phpunit
```
